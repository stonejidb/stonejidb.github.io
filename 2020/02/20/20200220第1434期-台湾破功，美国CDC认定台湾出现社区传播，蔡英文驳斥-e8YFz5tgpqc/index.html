大家好，今天是2020年的2月20号星期四，台湾最后也破功了，美国的疾病控制预防中心 c DC20日今天认定日本、新加坡、韩国、台湾、泰国及越南等地已经出现了明显的社区传播情形，但是尚未达到旅游警示的标准。台湾总统蔡英文今天20日驳斥，说台湾目前都是零星个案，并不符合社区传播的要件。怎么回事？台湾最近刚刚确诊第24例，也就是19号夜里昨天夜里的时候确诊的。

第24例是一个女士，她两年无旅游史感染源不明，所以就认为这应该属于社区传播了。蔡英文目前认为仅有这一例是传染源不明，所以他认为美国 c DC不应该把台湾和确诊的104例，还有的他的韩国，还有确诊了86例的日本和85例新加坡等6国共同列入出现明显社区传播的名单。

蔡英文对此不满，不过cDC是这样认为，就是说你有一个和有100个他认为也差不多是吧？这个意思是一样的，都算是社区传播，蔡英文觉得我只有一个，怎么说我觉得我之前其实我也说过，对于这个来讲，其实蔡英文是有一定的功劳的，由于蔡英文上台之后参与2016年，我记得2016年5月份他上台了之后，跟中国关系处理的就是非常的不好，非常不好的之后，中国大面积的减少了来台湾旅游的人数，当时来看是不好的，给台湾的经济带来了很大的冲击，使得很多的台湾的观光旅游业这些业者的收入都相应都减少了。

但是事实看来有一定的好处，好处之后就是说你看这一次看起来从日本、韩国、新加坡、台湾、越南等地比起来，台湾就是受到冲击最小的。之前在今年1月份选举之前，我也发了一篇发了一个视频对吧？我也说支持了大家，就是说选蔡英文。当时我也说我说看起来你选国民党可能跟中国交流更多，看起来好像中国人傻钱多，帮你卖水果还来人来你旅游买你的东西帮你发展经济。我说但是有一点你要明白，就是说自49年中国见证以来，你会发现在全世界的所有这些国家里，跟美国走的近的都比较兴旺发达安全跟中国走的近的都跟北朝鲜似的说是所以当时我也认为实际上你看起来好像你跟中国大陆交往多了之后，好像你能够多挣一点钱，有一些经济利益，但是实际上你得到的坏处是那种看不看不见摸不着的，他最终会给我觉得会带来很多问题，现在我想也印证了我这个说法。

另外一点作为一个就是说我觉得曾经一个在中国大陆生活过的人，作为一个比如比我觉得无论如何算一个华人，我觉得大陆出生的华人，我觉得在这里也应该像台湾，像日本、像新加坡，像韩国表达一个歉意，我觉得我们大陆政府大陆国还有国家政府人民都缺少了这种责任感，我们不认为就是说我们给别人添麻烦，我们觉得好像我们现在也这样，我们也都这么惨了，你不能这么想，你惨的话是你自己造成的对吧？

你给别别人惨的跟别人不能怪别人对吧？还是得怪你。所以我觉得从一个责任感来讲，我们应该感觉我们的的确确我们给周边的这些国家带来了麻烦，而且带来这种麻烦，其实主要还是因为我们的政府缺少贡献力，公信力缺少透明度，如果我们的政府有更多的公信力和透明度的话，这一次事件我相信不会是发展成现在这个样子。
